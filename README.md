ChessClient
========

A rich chess client designed to demonstrate the chessAPI.

Set up
=========

install dependencies:

(requires ruby for sass gem)

```sh
npm install
npm install http-server -g
npm install bower -g
bower install
gem install sass
```

build into dist folder:
```sh
gulp build
```

Start server
==========

```
http-server . # (in dist after gulp build)
```
