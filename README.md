ChessClient
========

A rich chess client designed to demonstrate the chessAPI at http://dxc4.com:8080
Demo: http://dxc4.com/

Set up
=========

install dependencies:

(requires ruby for sass gem)

```sh
npm install
npm install bower -g
bower install
gem install sass
```

build into dist folder:
```sh
gulp
```

Start server
==========

```sh
node server.js
```
